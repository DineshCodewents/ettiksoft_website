EV Components Test System

We make custom Q test systems for vehicle production lines.

Versatile Testing Solutions

EV Components tester is used in EV production line to test if the received components from the supplier are as per the specification. It does the following test automatically.

Key Features:

  • Battery charging and discharging current
  • Battery load and no load test
  • Electric Motor RPM and Current Test
  • Throttle control test
  • Charger voltage and current test
  • MCU test
  • Pedal test
